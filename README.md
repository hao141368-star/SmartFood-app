# 🌿 SmartFood - E-commerce App

Ứng dụng mua sắm thực phẩm sạch trực tuyến với giao diện hiện đại.

## 🚀 Cách cài đặt

### 1. Clone hoặc tải project
```bash
git clone https://github.com/hao141368-star/smartfood-app.git
cd smartfood-app
```

### 2. Cài đặt dependencies
```bash
npm install
```

### 3. Chạy project
```bash
npm start
```

Ứng dụng sẽ mở tại: http://localhost:3000
